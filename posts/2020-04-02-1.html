Сервис systemd для noip2.

Systemd Service for no-ip.com dynamic ip updater
Posted in: soft, systemd.

Создаём в /etc/systemd/system сервис с именем noip2.service:

[Unit]
Description=No-ip.com dynamic IP address updater
After=network.target
After=syslog.target

[Install]
WantedBy=multi-user.target
Alias=noip.service

[Service]
# Start main service
ExecStart=/usr/local/bin/noip2
Restart=always
Type=forking

Обновляем сервисы, ставим новый сервис в автозагрузку, запускаем сервис:

$ sudo systemctl daemon-reload
$ sudo systemctl enable noip2
$ sudo systemctl start noip2
