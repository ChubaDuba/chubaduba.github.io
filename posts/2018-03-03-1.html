ThreeColumns - отличная компоновка окон xmonad для широких экранов!

Компоновка окон XMonad.Layout.ThreeColumns для xmonad
Posted in: xmonad.

Если вы используете xmonad и у вас в хозяйстве есть ultrawide монитор, то компоновка окон XMonad.Layout.ThreeColumns точно будет одной из самых востребованных.
Чтобы установить компоновку, вносим изменения в xmonad.hs:

import XMonad.Layout.ThreeColumns
...
myLayouts = ThreeColMid 1 (3/100) (1/2) ||| ...
...
myConfig = defaultConfig
         { ...
         ...
         , layoutHook = myLayouts
         ...
         }

Можно использовать другой вариант компоновки: ThreeCol 1 (3/100) (1/2)
